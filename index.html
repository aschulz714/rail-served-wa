An Industry First:

Database and Web Map of Rail-Served Properties in Washington State

Go To Map

About This Project

This project provides an industry first database of rail-served properties in the state of Washington; spanning two Class I railroads, two regional railroads and sixteen short line railroads in the state. The results were visualized in an interactive web map, which can be accessed above. A rail-served property is one in which a rail spur extends on to a private property to allow for loading and unloading of rail cars.

The state of Washington provides a rich study area for this project. Among the 49 U.S. states with railroads, Washington ranks 16th in total track miles and 5th in total number of rail segments which transverse private property. These rail segments are the candidates for inclusion in the rail-served property database. Rail-served properties were identified using the U.S. Bureau of Transportation Statistics North American Rail Network (NARN) layer and a statewide parcel layer for Washington state.

Results

671 unique, rail-served properties were identified, representing a wide range of industry in Washington state. By count, agricultural properties accounted for the highest total of rail-served properties at 24%. Marine terminals on the other hand -- with just a handful of properties -- generated the most railcar loads in 2019, according to the Association of American Railroads.

Objectives

  • Use Python to automate the identification of rail-served properties
  • Create a web map with Leaflet to display railroads and rail-served properties
  • Create popups for each property with basic information
  • Symbolize rail-served properties by commodity type with custom SVGs (Scalable Vector Graphics)
  • Present land cover statistics from the USDA Cropland Data Layer

Interesting Rail-Served Properties

Railroads serve the Bangor Trident Base on the Olympic Peninsula; one of only two, Navy Trident submarine bases in the U.S.. Railroads also serve the Boeing Manufacturing Plant in Everett, WA (as shown above) via a rail spur with the steepest railroad grade in North America.

USDA Cropland Data Layer

Land cover statistics exported from the Cropland Data Layer web app show the 10 most common classes in Washington. The Evergreen State has a large number of rail-served properties in the lumber industry. Wheat is another valuable crop exported from Washington with wheat fields extending across the Palouse region of Eastern Washington. All rail-served properties in this area consist of grain elevators.
